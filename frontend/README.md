# Frontend (TypeScript)

Browser client for the 3D cube-surface multiplayer snake game.

## Prereqs

- Node.js 18+ recommended

## Install

```bash
cd frontend
npm install
```

## Run (dev)

```bash
cd frontend
npm run dev
```

Then open the printed URL and connect to the backend at `http://localhost:8000`.

## Intro video asset (optional)

To show an intro video before the lobby/login overlay, place your MP4 at:

- `frontend/public/assets/video/intro.mp4`
